# NATO Alphabet Converter

**NATO Alphabet Converter** is a browser-based tool designed to convert text into the German NATO phonetic alphabet (DIN 5009). It provides real-time conversion as you type, making it easy to spell out words, names, or complex terms over the phone or radio.

## Features

- **Real-time Conversion**: Instantly displays the NATO phonetic alphabet equivalent as you type.
- **German Standard (DIN 5009)**: Uses the official German spelling alphabet including special characters (Ä, Ö, Ü, ß).
- **Number Support**: Automatically converts digits 0-9 to their German word equivalents (Null, Eins, Zwei, etc.).
- **Special Character Handling**: Gracefully handles spaces and unsupported characters with clear indicators.
- **Clean, Modern UI**: Built with a premium gradient design and smooth transitions for an excellent user experience.
- **Single File Portability**: The entire tool is contained within a single HTML file (`nato-converter.html`), making it easy to share and deploy.

## Usage

1. Open `nato-converter.html` in any modern web browser.
2. Type any word, name, or text into the input field.
3. The NATO phonetic spelling will appear instantly below in an easy-to-read format.
4. Each letter is shown alongside its corresponding NATO word (e.g., "B = Berta").

## Technical Details

- **Stack**: Vanilla HTML, CSS, and JavaScript. No external dependencies or build steps required.
- **Alphabet Standard**: Implements the complete German DIN 5009 standard including all umlauts and special characters.
- **Privacy**: All processing happens locally in your browser. No data is sent to any server.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.
